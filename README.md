# ModelPirate Project Page

GitHub Pages-ready project page for **An Empirical Study on the Resilience of Partial Merging to Model Clone Attacks**.

The page follows the same academic project-page structure as DualTAP: title hero, author list, buttons, Abstract, Overview, Method, Main Results, Benchmark Summary, and BibTeX.

## Update before publishing

In `index.html`, replace the GitHub button if needed:

```html
<a class="btn secondary" href="https://github.com/wutt6678/ModelPirate">GitHub Code</a>
```

Also replace `assets/pdf/paper.pdf` with your final camera-ready PDF if needed.

## GitHub Pages deployment

1. Create a repository, e.g. `ModelPirate`.
2. Upload these files to the repository root.
3. Go to **Settings → Pages**.
4. Use either **Deploy from branch** (`main`, `/root`) or the included GitHub Actions workflow.
5. The page will be available at `https://<username>.github.io/<repo>/`.
